
# hopf-vibration

three.js visualization of the hopf vibration

## how

### install

```
git clone https://github.com/data-doge/hopf-vibration
cd
npm install
```

## develop

```
npm start
```

browse to <http://localhost:9966/>.

## test

```
npm test
```

## deploy

```
npm run deploy
```
